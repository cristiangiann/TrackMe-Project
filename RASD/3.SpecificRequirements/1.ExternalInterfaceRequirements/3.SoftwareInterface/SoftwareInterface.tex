\subsection{Software Interfaces}
			Data4Help will expose an API to \ic{Services} offered by \ic{Third parties} to send \ic{User data} and \ic{Group data}. In particular, its main functions are to:
			\begin{itemize}
				\item Retrieve all \ic{User data} available for a given \ic{Username}.
				\item Retrieve last \ic{n} records of \ic{User data} for a given \ic{Username}, where \ic{n} is a specified quantity.
				\item Retrieve all \ic{User data} available to form \ic{Group data} according to the following parameters: age, sex, residence.
				\item Send a notification to a specific \ic{User}.
			\end{itemize}
