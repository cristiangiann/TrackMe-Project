\subsection{Software Interfaces}
			Data4Help will expose an API to \ic{services} offered by \ic{third parties} to send \ic{user datas}. In details, it will enable to:
			\begin{itemize}
				\item Retrieve all records available for a given \ic{username}
				\item Retrieve last n records for a given \ic{username}, where n is a specified quantity
				\item Retrieve all records available for all people using the given service filtered according to ....\\ \todo{Decidere quali sono i modi possibili di filtrare i dati collettivi}
				\item Request the specified user authorization for subscribing to data updates.
				\item Unsubscribe the service from the data updates of a given user.
			\end{itemize}
