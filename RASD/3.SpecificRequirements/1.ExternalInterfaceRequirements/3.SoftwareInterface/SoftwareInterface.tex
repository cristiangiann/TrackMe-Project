\subsection{Software Interfaces}
			Data4Help will expose an API to \ic{services} offered by \ic{third parties} to send \ic{user datas}. In details, it will enable to:
			\begin{itemize}
				\item Retrieve all \ic{User data} available for a given \ic{Username}.
				\item Retrieve last \ic{n} records of \ic{User data} for a given \ic{Username}, where \ic{n} is a specified quantity.
				\item Retrieve all \ic{User data} available to form \ic{Group data} according to the following parameters: age, sex, \ic{Geographical area}.
				\item Request the specified user authorization for subscribing to data updates.
				\item Unsubscribe the service from the data updates of a given user.
			\end{itemize}
